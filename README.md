# Eutrobot Systems Website

A modern web application built with **Streamlit** to showcase **Eutrobot Systems**, its mission, robotic prototypes, team, and ongoing innovations in AI-driven environmental robotics.

## Features

* Modern responsive interface
* Interactive prototype showcase
* Project information pages
* Team and contact section
* Media gallery
* Built entirely with Streamlit

## Tech Stack

* Python
* Streamlit
* HTML & CSS

## Getting Started

Install the required package:

```bash
pip install streamlit
```

Run the application:

```bash
streamlit run app.py
```

## Purpose

This website serves as the official showcase for **Eutrobot Systems**, highlighting its vision, robotic prototypes, and commitment to developing innovative technologies for environmental sustainability.
